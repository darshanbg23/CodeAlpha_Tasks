# CodeAlpha Machine Learning Internship Tasks

This repository contains the projects completed during the **Machine Learning Internship at CodeAlpha**.
The goal of the internship was to apply machine learning concepts to real world problems by building complete projects involving **data preprocessing, model training, evaluation, and prediction systems**.

Each project focuses on a different machine learning application and demonstrates the use of **Python based ML and deep learning tools**.

---

# Project Descriptions

## 1. [Credit Score Prediction](https://github.com/darshanbg23/CodeAlpha_Tasks/tree/main/CodeAlpha_CreditScoringModel)

A machine learning model that predicts a user's **credit score category** based on financial and behavioral attributes.

**Key Features**

* Data preprocessing and feature engineering
* Training multiple machine learning models
* Model evaluation and accuracy comparison
* Prediction system for new user financial data

**Machine Learning Concepts Used**

* Classification models
* Feature scaling
* Model evaluation metrics

---

## 2. [Handwritten Character Recognition](https://github.com/darshanbg23/CodeAlpha_Tasks/tree/main/CodeAlpha_HandwrittenCharacterRecognition)

A deep learning based system that recognizes **handwritten digits and alphabet characters** from images.

The project trains **Convolutional Neural Networks (CNNs)** using two popular datasets:

* MNIST (digits)
* EMNIST (letters)

**Key Features**

* Image preprocessing pipeline
* CNN based deep learning model
* Digit and character prediction from input images
* Command line interface for training and prediction
